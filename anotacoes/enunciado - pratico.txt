DETALHES DO TESTE
Você deverá criar uma aplicação simples em PHP, Laravel e Mysql que consuma uma API pública e exiba os dados de forma organizada em uma página web.

    Escolha da API: Utilize a API pública abaixo. 
        OpenWeatherMap (para dados meteorológicos)

    Requisitos Funcionais:
        Crie uma rota no Laravel que acione uma função para buscar dados da API selecionada.
        Armazene os dados recuperados em uma tabela no banco de dados MySQL.
        Crie uma interface simples que mostre os dados armazenados. Pode ser uma tabela HTML simples.
        (Opcional) Permitir que o usuário filtre ou ordene os dados exibidos na interface.

    Estrutura do Banco de Dados:
        Defina a estrutura da tabela no MySQL para armazenar os dados. A estrutura deve estar normalizada e seguir boas práticas de design de banco de dados.
        Crie as migrations necessárias para gerar as tabelas.

    Validação e Tratamento de Erros:
        A aplicação deve tratar erros de conexão com a API e fornecer mensagens de erro amigáveis.
        Implementar validação básica para assegurar que os dados da API sejam armazenados corretamente no banco.

    Instruções para Execução:
        Forneça um README com instruções sobre como configurar e executar o projeto localmente, incluindo as dependências necessárias.

ENTREGÁVEIS

    Código-fonte do projeto em um repositório Git (GitHub, GitLab, etc.).
    Um README.md explicando como configurar e rodar o projeto.
